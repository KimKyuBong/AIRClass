# AIRClass GUI 관리 도구

교사분들을 위한 간단하고 직관적인 GUI 관리 앱입니다.

## 🎨 주요 기능

### ✅ 서버 관리
- **원클릭 시작/중지** - 버튼 하나로 서버 제어
- **실시간 상태 모니터링** - Docker 및 서버 상태 자동 확인
- **클러스터 정보** - 실행 중인 노드 수, 수용 가능 인원 표시

### ⚙️ 간편한 설정
- **초기 설정 마법사** - IP 주소 자동 감지
- **설정 변경** - 언제든지 IP, 비밀번호 수정 가능
- **자동 저장** - .env 파일 자동 생성 및 관리

### 📊 실시간 로그
- **서버 로그 뷰어** - docker-compose 출력 실시간 확인
- **타임스탬프** - 각 로그에 시간 표시
- **자동 스크롤** - 최신 로그 자동 표시

### 🌐 빠른 접속
- **선생님 페이지** - 원클릭으로 브라우저 열기
- **학생 페이지** - 학생용 URL 즉시 접속
- **관리자 페이지** - 클러스터 상태 모니터링

## 💻 설치 및 실행

### 방법 1: run.sh로 실행 (권장)

프로젝트 루트에서 실행. 의존성 자동 설치 후 GUI가 뜹니다.

```bash
cd /Users/hwansi/Project/AirClass   # 프로젝트 루트로 이동
./gui/run.sh
```

### 방법 2: 프로젝트 .venv로 직접 실행

**반드시** 프로젝트의 `.venv`에 설치하고, **같은 .venv의 python**으로 실행하세요.

```bash
cd /Users/hwansi/Project/AirClass
.venv/bin/pip install -r gui/requirements.txt
.venv/bin/python gui/airclass_gui.py
```

`source .venv/bin/activate` 후 `python gui/airclass_gui.py`만 쓰면 시스템/다른 venv를 쓸 수 있어 `ModuleNotFoundError`가 날 수 있습니다.

### 방법 3: 실행 파일 빌드

#### Windows
```cmd
cd gui
build-windows.bat
```
생성된 `dist/AIRClass.exe`를 프로젝트 루트 폴더에 복사 후 실행

#### macOS
```bash
cd gui
./build-macos.sh
```
생성된 `dist/AIRClass.app`을 Applications 폴더로 이동 후 실행

## 🛠️ 기술 스택

- **CustomTkinter** - 모던한 다크 모드 UI
- **tkinter** - Python 기본 GUI 라이브러리
- **python-dotenv** - 환경 변수 관리
- **requests** - API 통신
- **docker** - Docker 제어
- **PyInstaller** - 실행 파일 빌드

## 🎯 사용 방법

### 1. 최초 실행
1. GUI 실행 (`python airclass_gui.py`)
2. 초기 설정 마법사가 자동으로 표시됨
3. IP 주소 확인 (자동 감지됨)
4. 클러스터 비밀번호 입력
5. "저장하고 시작" 클릭

### 2. 서버 시작
1. "▶ 서버 시작" 버튼 클릭
2. 로그 창에서 진행 상황 확인
3. "✅ 서버가 성공적으로 시작되었습니다!" 메시지 대기
4. 15초 후 자동으로 상태 업데이트

### 3. 학생들에게 공유
1. "👨‍🎓 학생 페이지" 버튼 클릭하여 URL 확인
2. 학생들에게 `http://서버IP:5173/student` 주소 공유
3. 선생님은 "👩‍🏫 선생님 페이지"로 접속

### 4. 수업 종료
1. "⬛ 서버 중지" 버튼 클릭
2. 로그 확인
3. GUI 종료

## 🔧 문제 해결

### Docker가 실행되지 않습니다
- Docker Desktop을 수동으로 실행해주세요
- 작업 표시줄/메뉴바에서 Docker 아이콘 확인
- 초록색이 될 때까지 대기

### 서버 시작에 실패합니다
1. 로그 창에서 오류 메시지 확인
2. Docker가 정상 실행 중인지 확인
3. 포트 충돌 여부 확인 (8000, 5173 등)
4. `stop.bat`/`./stop.sh` 실행 후 재시도

### 설정을 변경하고 싶습니다
1. "⚙️ 설정" 버튼 클릭
2. IP 주소 또는 비밀번호 수정
3. "✅ 저장" 클릭
4. 서버 재시작 필요 (중지 후 시작)

## 💡 팁

### 자동 시작 설정
**Windows**: 시작 프로그램에 AIRClass.exe 추가  
**macOS**: 시스템 환경설정 > 사용자 및 그룹 > 로그인 항목

### 원격 관리
GUI는 로컬 컴퓨터에서만 실행하고, 브라우저는 다른 기기에서도 접속 가능합니다.

### 여러 선생님이 같은 네트워크 사용
각 선생님마다 다른 클러스터 비밀번호를 사용하면 서로의 서버가 섞이지 않습니다.
